# 2026-03-18 — Green Dot Badge Notification + Smaller Dot Refactor

## Summary

Added toolbar icon badge notifications, then replaced the oversized `setBadgeText` API with canvas-based icon compositing for a smaller, subtler green dot (~15% of icon size) in the top-right corner.

## Changes

### Session 1: Badge Notification (initial implementation)
- Added `urlToTabId` Map, `setBadge`/`clearBadge`/`clearBadgeForUrl` using badge text API
- Popup sends `clearBadge` on showing complete/error results
- Tab lifecycle cleanup listeners

### Session 2: Smaller Dot via Canvas Compositing

#### `extension/background.js`
- Replaced `setBadgeText`/`setBadgeBackgroundColor` with `setIcon({ imageData })` using canvas compositing
- Added `createDotIcon(size, color)` — loads icon PNG, draws small dot overlay, caches `ImageData` by `"size-color"` key
- `setBadge` is now async — composites 16px and 32px icons in parallel via `Promise.all`
- `clearBadge`/`clearBadgeForUrl` now async with `await`
- Added `clearBadgeForTab(tabId)` — clears badge and removes all URL→tab map entries for that tab
- `clearBadge` message handler now supports both `tabId` and `url` params
- `tabs.onUpdated` listener is now async

#### `extension/popup/popup.js`
- Sends single `clearBadge` with `tabId` unconditionally on init (opening popup = user is looking at tab)
- Removed per-branch `clearBadge` calls from complete/error handlers (init, click, poll)

#### `tests/background.test.js`
- Replaced `browser.action` mock: `setIcon` + `_icons` tracking instead of `setBadgeText` + `_badges`
- Added `browser.runtime.getURL` mock
- Added `Image` and `document.createElement('canvas')` mocks for canvas compositing
- New tests: `clearBadgeForTab`, `clearBadge` by tabId message, `setIcon` API failure resilience
- Updated all badge assertions from `_badges[tabId].text`/`.color` to `_icons[tabId].imageData`/`.path`

#### `tests/popup.test.js`
- Updated clearBadge tests: checks `tabId` (not URL), sent unconditionally (even for running)
- Poll no longer sends additional clearBadge (init handles it)

- popup.test.js: 55 passed, 0 failed
