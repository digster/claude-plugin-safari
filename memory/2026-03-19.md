# 2026-03-19 — Export Cached Summaries as Markdown

## Summary

Added export functionality to the Cache Explorer page — individual results as `.md` files and bulk export as `.zip` archives.

## Changes

### Session 1: Export Cached Summaries as Markdown

#### `extension/lib/jszip.min.js` (NEW — vendored)
- JSZip v3.10.1 (~97KB minified) for creating zip archives in the browser
- Loaded via `<script>` tag before `cache-explorer.js`

#### `extension/cache-explorer/cache-explorer.html`
- Added `<script src="../lib/jszip.min.js">` before app script
- Added "Export All as Zip" icon button (`#export-all-btn`) with download SVG in sidebar header (before Refresh)
- Added "Export" button (`#detail-export-btn`) between Copy and Delete in detail actions

#### `extension/cache-explorer/cache-explorer.js`
- Added `downloadFile(content, filename, mimeType)` — Blob + anchor click download helper, handles both string and Blob inputs
- Added `hostnameFromUrl(url)` — extracts hostname from URL for filenames, falls back to `'unknown-url'`
- Added `formatResultAsMarkdown(item)` — formats cached result as markdown with `# URL` heading, metadata table (status, prompt, timestamps, duration, cost, tokens in ISO format), `---` separator, and response content. Handles edge cases: error, cancelled, empty responses
- Added single export listener — exports `selectedResult` as `claude-<hostname>.md`, shows "Exported!" feedback
- Added bulk export listener — uses JSZip to create zip with deduplication suffix for same-hostname URLs, downloads as `claude-cache-export-YYYY-MM-DD.zip`

#### `project.pbxproj`
- Registered `extension/lib/` folder in Xcode project (PBXBuildFile, PBXFileReference, PBXGroup, PBXResourcesBuildPhase) — IDs `86D77A9F2F6901BF009BEC45` and `86D77AA02F6901BF009BEC45`

#### `tests/cache-explorer.test.js` (NEW)

- Tests edge cases: error/cancelled/empty results, null/invalid URLs, unwrapped results, duration_ms fallback, response.error fallback
